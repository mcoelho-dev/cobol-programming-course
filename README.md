# COBOL Programming with VSCode — Lab Exercises

Repository with the completed lab exercises from the **[Open Mainframe Project COBOL Programming Course](https://github.com/openmainframeproject/cobol-programming-course)**, taken through the interactive labs platform (ibm.biz/cobollabs).

The course teaches COBOL development in a mainframe (z/OS) environment using VS Code, Zowe Explorer, and JCL, covering everything from environment setup to compiling and running real COBOL programs on a z/OS system.

## Environment

- **Editor**: VS Code with Zowe Explorer
- **System**: z/OS (Open Mainframe Project lab server)
- **Userid**: Z76689
- **Languages**: COBOL, JCL
